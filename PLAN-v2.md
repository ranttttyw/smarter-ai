# Starry v2 · 星球漫游 升级计划

## 产品愿景

从"AI 对比工具"进化为"AI 宇宙探索体验"。

用户不是在选工具，而是在一片星空中找到最懂自己的那颗星球。每次对话都是一次星际旅行，每次选择都在描绘你的星际人格画像。

---

## Phase 1: 动态宇宙 (视觉引擎)

**目标：让页面活起来**

### 1A. Canvas 星空背景
- 用 `<canvas>` 替换当前 CSS 伪元素星空
- 粒子系统：200-300 个星星，大小 / 亮度 / 闪烁频率随机
- 缓慢漂移运动（视差效果），鼠标移动时产生轻微视差偏移
- 偶尔出现流星拖尾动画
- 深空渐变底色保留（#0a0e27 → #1a0f3d → #0d1b4a）

### 1B. 星球漂浮系统
- 3 个模型 = 3 颗星球，用 CSS/Canvas 绘制
- 每颗星球有独特视觉：
  - **Gemini** → 蓝色水晶星球，表面有光泽纹理
  - **DeepSeek** → 紫色星云星球，带光环
  - **GLM** → 青绿极光星球，表面有极光流动
- 星球在宇宙中缓慢漂浮（正弦/余弦轨道）
- 星球周围有微光晕 + 小型卫星/碎片环绕
- 星球大小 ~80-100px，间距自适应屏幕宽度

### 1C. 交互反馈
- **Hover**：星球微微放大 + 光晕增强 + 显示模型名
- **选中**：星球亮起，发出脉冲光波（ripple 效果）
- **未选中**：星球变暗变小，退入背景
- **正在加载（思考中）**：选中的星球自转加速 + 轨道光环旋转

## Phase 2: 降落动画 (Deep Chat 体验)

**目标：选择继续聊天 = 降落在这颗星球上**

### 2A. 降落过渡动画
- 点击"继续聊天"触发：
  1. 其他星球渐隐飞散
  2. 目标星球开始放大（scale 1 → 全屏）
  3. 视角"穿过"星球表面（zoom + blur 过渡）
  4. 星球颜色成为聊天界面的主题色
- 总时长 ~1.2s，CSS animation + requestAnimationFrame

### 2B. 星球聊天室
- 聊天界面背景 = 星球表面纹理（微妙渐变）
- 顶栏显示："🪐 已降落在 [Model Name] 星球"
- 退出聊天 = "离开星球"动画（缩小 + 回到宇宙视角）

## Phase 3: 星际评论系统 (Comment on Answers)

**目标：让用户对每个回答发表看法**

### 3A. 评论 UI
- 每个模型回答下方新增一排交互：
  - ⭐ 快速评分（1-5 星，点击星星）
  - 💬 写评论（点击展开文字输入框，最长 200 字）
  - 快捷标签芯片：`太啰嗦` `很有创意` `跑题了` `正合我意` `太死板` `有启发`
- 评论后显示在回答下方，带时间戳
- 数据存入 localStorage（后续可同步到后端）

### 3B. 数据结构
```json
{
  "questionId": "q_1234567890",
  "question": "人生的意义是什么",
  "questionType": "philosophical",  // 系统自动分类
  "timestamp": 1234567890,
  "responses": {
    "deepseek": {
      "rating": 4,
      "comment": "说到我心坎里了",
      "tags": ["有启发", "正合我意"],
      "chosen": true  // 是否选择继续聊天
    },
    "gemini": {
      "rating": 2,
      "comment": "",
      "tags": ["太死板"],
      "chosen": false
    }
  }
}
```

## Phase 4: AI 人格画像 (Persona Profiling)

**目标：根据用户的选择和评论，描绘出"星际人格"**

### 4A. 问题自动分类
- 用 DeepSeek 对用户的提问做轻量分类（或前端规则匹配）
- 分类维度：
  - **实用型**：写邮件 / 写文案 / 工作相关
  - **创意型**：写小说 / brainstorm / 创作
  - **哲学型**：人生意义 / 价值观 / 深度思考
  - **情感型**：安慰 / 倾诉 / 关系
  - **知识型**：学习 / 解释 / 分析

### 4B. 偏好分析引擎
- 统计维度（从评论 + 选择数据中提取）：
  - 各类问题分别偏好哪个模型
  - 高评分回答的共同特征（哪些 tags 反复出现）
  - 用户在什么场景倾向于 deep chat
  - 评论中的关键词聚类

### 4C. 人格画像展示 — "你的星际护照"
- 一个精美的个人页面 / 卡片：
  - **星际旅行者类型**：比如 "理性探索者"、"浪漫哲学家"、"效率至上主义者"
  - **星球亲和度雷达图**：Gemini 30% / DeepSeek 55% / GLM 15%
  - **决策风格**：
    - "你在深度思考时偏好 DeepSeek 的思辨角度"
    - "你做实用任务时偏好 Gemini 的简洁风格"
  - **标签云**：你最常给出的评价标签
  - **旅行统计**：总对话数 / 降落过星球 / 最长对话
- 画像可以截图分享（生成卡片图）

### 4D. 数据存储
- Phase 1: 全部 localStorage
- Phase 2 (future): 后端 SQLite/JSON 持久化，支持登录同步

## Phase 5: Presets 升级

**目标：匹配新的双场景定位**

### 5A. 新 Preset 体系
- **写作星域** ✍️
  - "帮我写一篇小红书种草文案，主题是「一个人去京都旅行」"
  - "给我写一封拒绝 offer 的邮件，语气委婉但坚定"
  - "帮我写一段产品介绍，一款帮助学生备考的 AI 工具"
- **思想星域** 💭
  - "如果你只剩一年生命，你会怎么度过？"
  - "自由和安全之间，你更看重哪个？"
  - "你觉得孤独是一种力量还是一种缺失？"

### 5B. 分类 Tab 切换
- 输入框上方两个 tab：✍️ 写作 | 💭 思想
- 切换 tab 显示不同的 preset 建议
- 或者做成一个"星域导航"，点击不同星域展示对应预设

## 面试叙事框架

> "Starry 不只是让你比较 AI 模型——它是帮你找到最懂你的 AI 思考伙伴。"

**痛点**：市场上有很多模型，用户不知道哪个最适合自己，而且"适合"是分场景的。

**解决方案**：
1. **Explore** — 同一个问题，多个模型并排回答，像在宇宙中巡游
2. **Evaluate** — 不只看结果，还能评论、打分、标记
3. **Understand** — 系统分析你的选择偏好，构建你的 AI 人格画像
4. **Land** — 找到最对味的那个，深入对话

**产品指标**（PM 面试加分）：
- Engagement: 平均每次使用的对话轮数
- Stickiness: 7日留存率
- Conversion: 从"浏览模式"到"降落深聊"的转化率
- Discovery: 用户尝试不同模型的多样性指数

**技术亮点**：
- 多 LLM 并行调用架构
- 轻量级 meta-analysis（用模型分析模型）
- 前端 Canvas 粒子系统
- 行为数据驱动的人格分析
